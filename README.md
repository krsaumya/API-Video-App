# 🎬 Video Streaming Application

A full-stack video streaming platform built with **Flask (Backend)** and **Vite + React (Frontend)**.  
The application follows a **thin client architecture**, where all business logic resides in the backend and the frontend acts as a presentation layer.

---

## 🚀 Tech Stack

### Frontend
- React (Vite)
- HTML5
- CSS3
- JavaScript
- Axios

### Backend
- Flask
- Flask-CORS
- RESTful APIs
- JWT Authentication
- SQLite / PostgreSQL (configurable)

---

## 📌 Features

- User Authentication (Login / Register)
- Secure JWT-based Authorization
- Video Upload Functionality
- Video Streaming Support
- Role-based Access Control
- REST API-driven Architecture
- Thin Client Design (Business logic handled in backend)

---

## 🏗️ Architecture Overview

- The **React frontend** acts as a thin client.
- All authentication, validation, and streaming logic is handled by the **Flask backend**.
- Communication between frontend and backend is done using REST APIs.
- Secure token-based session management.
